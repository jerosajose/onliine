Jokes aside

This website has been created using HTML, CSS, JavaScript and jQuery, with the help of:

wii.js.org by @andrewplus

The main inspiration and motivation to build this site. Even though they look very similar, this has been made from scrath, using all the UI assets at his project with some tweaks, mainly to have an easy start with the site.

Now it's sort of abandoned, but it still runs great, check him out on GitHub.

The Spriters Resource – spriters-resource.com/wii

It's an amazing site with a really enthusiastic community. Pretty much all the images left to complete the site, that weren't on andrew's site, have been taken from there. All the content that peopled uploaded there plus some CSS animations made possible the main menu channel arts.

Dolphin Emulator & YouTube tour videos

Another great source of inspiration to know how is exactly everything is styled. System audio effects have also been obtained through YouTube and SoundCloud.

EZGIF – ezgif.com

Really, really handy website for whatever thing you wanna do with GIFs. It's been mainly used with video conversion and some trickery for channel banners videos.

Fontworks – lets.fontworks.co.jp

The original authors of the famous Rodin font used on both Wii and Wii U.

Other tools used

Gyazo (for video (banner) capture), Nintendo's web (Mii Studio), GIMP (image editing), Inkscape (logo design)
